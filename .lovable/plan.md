

# Карта Пути — The Circuit Board

## Экран
Единственный экран: список воронок в стиле «печатной платы» — горизонтальные линии-проводники с узлами (кодовое слово, продукт).

## Структура

### 1. Шапка
- Заголовок: `Paths / Active (4)` — в стиле Geist Sans
- Панель фильтров: три сегментированных контрола (по кодовому слову / типу контента / продукту) с `blur-md` подложкой
- Кнопка «New Path» — `bg-foreground text-background`

### 2. Активные воронки (4 строки)
Каждая строка — flex-контейнер `h-16`:
- **[•••]** — круглая кнопка-триггер с тремя янтарными точками
- **Горизонтальная линия** — `1px`, проходит сквозь всю строку
- **Кодовое слово** — янтарный бейдж (`bg-primary text-black`, uppercase, `rounded`, `text-[11px]`)
- **Продукт** — белый текст + иконка стрелки, прижат вправо

Данные:
1. `кейс` → статья про кейс на 1.4 млн
2. `образ` → туториал как примерить любую причёску
3. `макс` → инструкция по переносу ТГ-канала в MAX
4. `система` → AI-команда под ключ (с пометкой трипваер)

### 3. Раскрытие воронки
При клике на [•••] — строка раздваивается, дочерние элементы появляются с `staggerChildren: 0.05s`. Вертикальная линия с градиентом от янтарного к прозрачному соединяет родителя с детьми. Дочерние строки — `h-12`, с отступом, `font-mono` для метрик.

### 4. Неактивные воронки (2 строки)
Та же структура, но `opacity-40` и `grayscale`. Без текста "Inactive" — только визуальное снижение контраста.

## Дизайн-система
- Фон: `#090909`
- Поверхность: `rgba(255,255,255,0.03)`
- Бордеры: `rgba(255,255,255,0.08)`
- Акцент: `#F59E0B` (янтарный)
- Шрифт: Geist Sans + Geist Mono для цифр
- Радиус: max `8px`
- Анимации: `cubic-bezier(0.2, 0.8, 0.2, 1)`, 300ms

## Mobile First
- На мобильных: строки становятся вертикальными стеками — триггер сверху, кодовое слово и продукт друг под другом
- Фильтры — горизонтальный скролл
